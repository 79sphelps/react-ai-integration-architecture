# React AI Integration Architecture

Production-grade AI patterns in React using free and local models.

This repository is a collection of mini-projects that demonstrate how to design, build, and scale AI-powered user experiences and not just call APIs.

---

## What This Repo Teaches

- AI-first UX patterns (streaming, copilots, human-in-loop)
- Prompt engineering with structured outputs
- Retrieval-Augmented Generation (RAG)
- Local AI execution (no paid APIs required)
- AI state management & caching
- Observability & debugging
- Provider abstraction architecture

---

## Why This Repo Exists

So many AI tutorials show:

> “Call this API and display text”

This repo focuses on:

> “How do you build real AI-powered products?”

---

## Architecture Principles

- No classes (functional + hooks only)
- Fully typed APIs
- Accessible UI (ARIA compliant)
- Testable systems
- Modular architecture
- Reusable AI primitives

---

## Testing Strategy

- Unit: hooks & utilities
- Integration: component behavior
- E2E: full AI flows
